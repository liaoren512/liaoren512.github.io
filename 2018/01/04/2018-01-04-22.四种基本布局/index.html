22.四种基本布局

发表于 2018-01-04 |
字数统计: | 阅读时长 ≈

线性布局

线性布局，一种非常常用的布局。
它所包含的控件在线性方向上依次排列。
android:orientation，指定排列方向水平还是垂直

<LinearLayout
    android:orientation="vertical"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

</LinearLayout>

android:layout_gravity，指定控件在布局中的对齐方式。
android:layout_weight，允许我们使用比例的方式指定控件大小。

<LinearLayout
    android:orientation="horizontal"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <EditText
        android:id="@+id/input"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_weight="1"
        android:hint="Type something" />

    <Buttong
        android:id="@+id/send"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_weight="1"
        android:text="Send" />
</LinearLayout>

EditText、Button的宽度都指定成了0dp，我们使用weight属性，
此时控件的宽度不再由layout_width来决定

<LinearLayout
    android:orientation="horizontal"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <EditText
        android:id="@+id/input"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_weight="1"
        android:hint="Type something" />

    <Buttong
        android:id="@+id/send"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Send" />
</LinearLayout>

只指定EditText的layout_weight属性，并把Button宽度设为wrap_content
这表示Button宽度仍然按照wrap_content来计算，EditText则占满屏幕剩余空间。
这种方式，适配性会更好。

相对布局

相对布局，通过相对定位的方式让控件出现在布局的任何位置。

<RelativeLayout
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/button1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentLeft="true"
        android:layout_alignParentTop="true"
        android:text="Button" />
    <Button
        android:id="@+id/button2"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentRight="true"
        android:layout_alignParentTop="true"
        android:text="Button" />
    <Button
        android:id="@+id/button3"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerInParent="true"
        android:text="Button" />
    <Button
        android:id="@+id/button4"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentBottom="true"
        android:layout_alignParentLeft="true"
        android:text="Button" />
    <Button
        android:id="@+id/button5"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentBottom="true"
        android:layout_alignParentRight="true"
        android:text="Button" />
</RelativeLayout>

控件除了相对于父布局进行定位，也可以相对于控件进行定位。

<RelativeLayout
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/button1"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerInParent="true"
        android:text="Button" />
    <Button
        android:id="@+id/button2"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_above="@id/button1"
        android:layout_toLeftOf="@id/button1"
        android:text="Button" />
</RelativeLayout>

android:layout_above，让一个控件位于另一个控件的上方。
android:layout_below，让一个控件位于另一个控件的下方。
android:layout_toLeftOf，让一个控件位于另一个控件的左侧。
android:layout_toRightOf，让一个控件位于另一个控件的右侧。

帧布局

所有的控件都会默认在布局的左上角。

<FrameLayout
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <TextiView
        android:id="@+id/text_view"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="This is TextView" />
    <ImageView
        android:id="@+id/image_view"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:src="@mipmap/ic_launcher" />
</FrameLayout>

文字、图片都位于布局的左上角。
ImageView是在TextView之后添加的，图片在文字上方。

百分比布局

只有LinearLayout支持使用layout_weight实现比例指定控件大小，
如果想用RelativeLayout实现两个按钮平分布局宽度效果，则比较困难。
为此，Android引入百分比布局

百分比布局，不再使用wrap_content、match_parent等方式、允许直接在控件布局中使用百分比。
因为LinearLayout已经支持按比例指定控件大小，
所以百分比布局只为FrameLayout、RelativeLayout进行了扩展。

  • 引入

    打开app/build.gradle，

    在dependencies闭包添加：compile ‘com.android.support:percent:24.2.1’

  • 修改activity_main布局文件
